## Atividade proposta

Escreva um programa em C++ que leia um número inteiro e mostre a 
tabuada da multiplicação conforme o formato abaixo. 

Por exemplo, se número = 8, deverá ser impresso:
```
8 x 0 = 0
8 x 1 = 8
8 x 2 = 16
......…
8 x 10 = 80
```
